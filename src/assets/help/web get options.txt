MDX WEB GET

  Use the 'mdx web get' command to fetch web pages and create markdown from their content.

USAGE: mdx web get "URL" ["URL2" [...]] [...]

OPTIONS

  BROWSER

    --browser BROWSER            Specify the browser to use (chromium, firefox, webkit)
    --interactive               Run browser in interactive mode (not headless)
    --wait-for SELECTOR         Wait for a specific CSS selector to be present on the page
                               (e.g. --wait-for "#main-content" or --wait-for ".article")

  CONTENT

    --get                       Get page content (default: false)
    --strip                     Strip HTML from page content (default: false)

  OUTPUT

    --save-folder FOLDER        Save page content to the specified folder
    --save-page-output FILE     Save page output to the specified template file
                               (e.g. {filePath}/{fileBase}-output.md)
