MDX WEB GET

  Use the 'mdx web get' command to fetch web pages and create markdown from their content.

USAGE: mdx web get "URL" ["URL2" [...]] [...]

EXAMPLES

  EXAMPLE 1: Get a web page and convert to markdown

    mdx web get "https://example.com" --get --strip

  EXAMPLE 2: Get multiple web pages and save to files

    mdx web get "https://example.com" "https://example.org" --get --save-folder pages/

  EXAMPLE 3: Wait for specific content to load before processing

    mdx web get "https://example.com" --get --wait-for "#main-content"
    mdx web get "https://example.com" --get --wait-for ".article" --interactive

  EXAMPLE 4: Use different browsers and save output

    mdx web get "https://example.com" --browser firefox --get --save-page-output "pages/{fileBase}.md"
