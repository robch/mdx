USAGE: mdx run [OPTIONS] commands [...] [@file] [@@glob]
  Run a shell command

    Commands can be provided on the command-line, via file (@file), 
    or via multiple files (@@glob). Each line is a command or part of 
    a multi-line script.

  OPTIONS:
    --cmd COMMANDS        Run using cmd.exe
    --bash COMMANDS      Run using bash
    --powershell COMMANDS Run using PowerShell
    -e, --env NAME=VALUE Set environment variable
    --save-output PATH   Save output to markdown file
